# DevOps Production Projects

This repository contains end-to-end DevOps projects demonstrating:

- Linux system administration
- Networking fundamentals
- Docker & containerization
- CI/CD automation using GitHub Actions
- AWS infrastructure deployment
- Nginx reverse proxy and SSL
- Monitoring with Prometheus and Grafana
- Kubernetes fundamentals

## Projects Overview

1. CI/CD Microservices (Python + Go)
2. GitHub Actions Automation
3. AWS Infrastructure Setup
4. Nginx Reverse Proxy with SSL
5. Monitoring and Alerting
6. Production Docker Compose
7. Kubernetes Basics

All projects are deployed and tested on Linux cloud servers.
